## 🔀 Local Environment Setup & Execution

To test or deploy any individual application module from within this monorepo tracking tree:

1. Clone this repository onto your workspace via SSH:
   ```bash
   git clone git@github.com:batyaboyo/mobile_app_development.git
   ```
2. Navigate directly into the target project namespace folder:
   ```bash
   cd mobile_app_development/UGTours_App
   ```
3. Fetch dependencies and initialize the specific developer framework configurations:
   ```bash
   # For Flutter/Dart projects:
   flutter pub get
   
   # For Node/React Native configurations:
   npm install
   ```
4. Fire up your emulator stream and boot the compiler layer:
   ```bash
   flutter run  # Or system-specific native execution flags
   ```
