# Sobriety - Native Android Sobriety Tracker

A clean, supportive, and private sobriety tracking application designed to help you stay motivated on your journey. Ported from the original web version to a native Android experience for better performance and integration.

## 🌟 Key Features

- **Live Streak Counter**: Track your sobriety down to the minute with a native live timer.
- **Financial Savings**: Automatically calculates money saved based on your previous weekly spend.
- **Daily Check-ins**: Log progress or slips, track mood emojis, and record urge intensity.
- **Private Journaling**: Keep local notes about your reflections and struggles.
- **Health Timeline**: Visualize your body's recovery with a detailed roadmap of medical benefits.
- **Coping Tools**:
  - **4-7-8 Breathing**: Native interactive animation to help ground you.
  - **Emergency Support**: 5-minute cooldown timer with quick access to support lines.
  - **Customizable Distractions**: Manage your own list of activities to navigate cravings.
- **Enhanced Privacy**: All data is stored locally on-device using Room (SQLite). No cloud, no tracking.
- **Material 3 Design**: Modern, beautiful UI with full Dark/Light mode support.

## 🛠️ Tech Stack

- **Kotlin**: Core language.
- **Jetpack Compose**: Declarative UI framework.
- **Material 3**: Design system and components.
- **Room Database**: Structured local data persistence.
- **DataStore Preferences**: Secure storage for user settings.
- **StateFlow & ViewModel**: Reactive state management.

## 🚀 Getting Started

1. **Prerequisites**: Install [Android Studio](https://developer.android.com/studio).
2. **Open Project**: Import the root folder `sobriety` (it contains the Gradle project).
3. **Build & Run**: Run on an emulator or physical device (API 26+).
4. **Onboarding**: Enter your quit date and weekly spend to begin your journey.

## 💾 Data Management

Import and Export functionality is available in the **Resources** tab, allowing you to backup your progress as a JSON file.
